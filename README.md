# Spotify Web Player Clone

This web project combines HTML, CSS, and JavaScript. I chose to clone the Spotify web interface to challenge myself with complex web layouts—specifically utilizing CSS Grid and Flexbox—and to learn how to structure a responsive webpage from the ground up without relying on any external frameworks.

The goal of this project isn't just to mimic the visual design, but to understand the fundamental code architecture behind modern web applications. So far, it features a pixel-perfect navigation bar with custom CSS tooltips, precise layout alignment, and smooth hover state transitions.

Tech Stack used:

HTML
CSS
JavaScript
